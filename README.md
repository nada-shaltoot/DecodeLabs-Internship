# Password Strength Checker

## Description
This project is a simple Password Strength Checker built using Python.

It checks the password based on:
- Password length
- Uppercase letters
- Numbers
- Special characters

The program classifies the password as:
- Weak
- Medium
- Strong

## How to Run

1. Open the project in VS Code.
2. Run the file:

```bash
python password_checker.py
```

3. Enter your password.
4. The program will display the password strength.
