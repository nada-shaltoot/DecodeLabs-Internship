DecodeLabs Cyber Security Internship Projects

This repository contains the projects completed during the DecodeLabs Cyber Security Internship (2026).

Projects

1. Password Strength Checker

A Python program that evaluates password strength based on:

* Password length
* Uppercase letters
* Numbers
* Special characters

2. Encryption & Decryption Tool

A Python application for encrypting and decrypting text using a simple encryption technique.

3. Phishing Awareness Analysis

A cybersecurity report explaining phishing attacks, common warning signs, prevention methods, and security best practices.

4. System Vulnerability Report

A vulnerability assessment report based on a personal system audit, including firewall verification, administrator account review, operating system checks, and security recommendations.
